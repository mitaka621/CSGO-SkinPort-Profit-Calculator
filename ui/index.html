CSGO Profit Calculator

Show description of the tool

This tool allows the user to select the price they are willing to pay and the rows they want to generate. After that by using the SkinPort API and the SteamFolio API the contents of both platforms (Steam and SkinPort) are extracted and then compared. The premise is that you buy an item from SkinPort and then sell it on Steam Market for higher price than you paid for!
